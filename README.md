# Nepal Population Data Analysis

A data analysis project using Python to analyze and visualize population data of top 10 cities in Nepal.

## Features

- Population comparison of top 10 Nepal cities
- Pie chart showing population distribution
- Province wise population breakdown
- Statistical summary (total, average, highest, lowest)

## Built With

- Python 3
- Pandas
- Matplotlib

## Charts Generated

1. Bar Chart - Top 10 Cities by Population
2. Pie Chart - Population Distribution
3. Bar Chart - Population by Province

## How to Run

1. Install required libraries:
   pip install pandas matplotlib
2. Run the script:
   python analysis.py
3. Charts will be displayed and saved as PNG files

## Key Findings

- Kathmandu has the highest population (1,003,285)
- Pokhara is second largest city (518,452)
- Bagmati Province dominates with 3 major cities
